# Solify
Solify is an educational Android application designed to help users learn music interactively. The app offers structured lessons, practical exercises, and progress tracking to make music learning engaging and effective.

### Figma Solify App:
https://www.figma.com/design/0ltaLQbzLH3EYv9Liy0VDH/Solify?node-id=0-1&p=f&t=j5vUxX3dvqvDPpWF-0

### Key Features:
- User Authentication – Secure registration and login system with session management
- Personal Profile – Customizable user profile with avatar upload
- Interactive Lessons – Step-by-step music theory lessons with practice materials
- Knowledge Tests – Quizzes to reinforce learning and track understanding
- Training Section – Practical exercises for skill development
- Progress Tracking – Monitor completed lessons, tests, and overall learning progress
- Intuitive Navigation – Easy-to-use bottom navigation bar for quick access between sections

### Tech Stack:
Language: Kotlin
UI Framework: Jetpack Compose
Architecture: Clean Architecture + MVVM
Dependency Injection: Hilt
Local Database: Room
Image Loading: Coil
Authentication: Custom session manager with DataStore
Navigation: Navigation Compose
